Structure de base d'une page HTML

Dans cette section, nous allons explorer la structure de base d'une page HTML. Chaque page HTML commence par une déclaration de type de document, suivie des balises HTML, head et body.

Structure de base d'une page HTML Une page HTML commence toujours par un doctype qui informe le navigateur du type de document à attendre. Ensuite, elle est composée de plusieurs balises essentielles :
  1. <!DOCTYPE html> : Déclare le type de document.
  2. <html> : Racine du document HTML.
  3. <head> : Contient les métadonnées (titre, encodage, liens CSS, etc.).
  4. <body> : Contient le contenu visible de la page (titres, paragraphes, images, etc.).
